  TELEGRAM CHAT PANEL - ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ НА VPS
═══════════════════════════════════════════════════════════════

ЧТО ВАМ НУЖНО СДЕЛАТЬ:

ЧАСТЬ 1: НА ВАШЕМ ЛОКАЛЬНОМ КОМПЬЮТЕРЕ
────────────────────────────────────────────────────────────────

1. Скачайте весь проект как ZIP архив из Emergent

2. Распакуйте архив в любую папку

3. Скопируйте следующие файлы В КОРЕНЬ распакованной папки:
   ✓ Dockerfile.backend
   ✓ Dockerfile.frontend  
   ✓ docker-compose.prod.yml
   ✓ nginx.conf
   ✓ install.sh
   ✓ .env.production

4. Скопируйте .env.production в папку frontend и переименуйте в .env

5. Откройте файлы и замените YOUR_VPS_IP на ваш реальный IP:
   - docker-compose.prod.yml (2 места)
   - frontend/.env (1 место)

6. Загрузите всю папку на VPS через SCP или FileZilla:
   
   Через SCP (из терминала Mac/Linux):
   scp -r /путь/к/папке root@ВАШ_IP:/root/telegram-app
   
   Через FileZilla (Windows/Mac):
   - Хост: sftp://ВАШ_IP
   - Пользователь: root
   - Порт: 22
   - Загрузите папку в /root/telegram-app


ЧАСТЬ 2: НА VPS СЕРВЕРЕ
────────────────────────────────────────────────────────────────

1. Подключитесь по SSH:
   ssh root@ВАШ_IP

2. Перейдите в папку:
   cd /root/telegram-app

3. Сделайте скрипт исполняемым:
   chmod +x install.sh

4. Запустите установку:
   ./install.sh

5. Когда скрипт спросит IP адрес - введите IP вашего VPS

6. Ждите 10-15 минут пока все установится

7. ГОТОВО! Откройте в браузере:
   http://ВАШ_IP

   Логин: admin
   Пароль: admin123


ПОЛЕЗНЫЕ КОМАНДЫ НА VPS:
────────────────────────────────────────────────────────────────

Посмотреть статус:
  docker compose -f docker-compose.prod.yml ps

Посмотреть логи:
  docker compose -f docker-compose.prod.yml logs

Перезапустить:
  docker compose -f docker-compose.prod.yml restart

Остановить:
  docker compose -f docker-compose.prod.yml down

Запустить:
  docker compose -f docker-compose.prod.yml up -d


ВАЖНО:
────────────────────────────────────────────────────────────────

1. Убедитесь что на VPS открыт порт 80:
   ufw allow 80/tcp

2. После первого входа смените пароль администратора!

3. Минимальные требования VPS:
   - 2GB RAM
   - 20GB диск
   - Ubuntu 24.04


ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
────────────────────────────────────────────────────────────────

1. Проверьте логи:
   docker compose -f docker-compose.prod.yml logs

2. Проверьте что все контейнеры запущены:
   docker compose -f docker-compose.prod.yml ps

3. Убедитесь что вы правильно заменили YOUR_VPS_IP на ваш IP

4. Проверьте что порт 80 открыт в firewall
